# Tic Tac Toe Game

A classic Tic Tac Toe game built with HTML, CSS, and vanilla JavaScript. Features a clean interface with customizable colors and animated winning lines.

## Features

- **Two Player Game**: Play X vs O with alternating turns
- **Win Detection**: Automatically detects wins in rows, columns, and diagonals
- **Animated Winning Line**: Visual line animation appears when a player wins
- **Draw Detection**: Recognizes when the game ends in a draw
- **Color Customization**: Change the game's color theme using the color palette menu
- **Turn Indicator**: Shows which player's turn it is
- **Reset Functionality**: Restart the game anytime with the reset button

## How to Play

1. Open `index.html` in your web browser
2. Player X starts first
3. Click on any empty box to place your mark
4. Players alternate turns between X and O
5. First player to get three marks in a row (horizontal, vertical, or diagonal) wins
6. Click the reset button to start a new game

## Customization

Click the gear icon to open the color palette menu and choose a different theme color for the game board.

## Game Rules

- The game is played on a 3x3 grid
- Player X always goes first
- Players take turns placing their marks (X or O)
- The first player to get 3 marks in a row wins
- If all 9 squares are filled with no winner, the game is a draw

## Technologies Used

- HTML5
- CSS3 (with animations and transitions)
- Vanilla JavaScript (ES6+)
- Font Awesome (for icons)
